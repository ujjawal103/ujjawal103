## 🚀 Flagship Project — TapResto

<div align="center">

[![TapResto](https://img.shields.io/badge/🍽️%20TapResto-Live%20in%20Production-00f5c4?style=for-the-badge&labelColor=0d1224)](https://tapresto.online)

</div>

> A full-featured **multi-tenant SaaS** digital ordering platform for restaurants.
> QR-code table ordering · Real-time order tracking · Admin dashboard · Menu management · Push notifications

| Feature | Stack |
|---|---|
| Frontend | React.js + Tailwind CSS |
| Backend | Node.js + Express.js |
| Database | MongoDB |
| Notifications | Firebase FCM |
| Hosting | AWS EC2 |
| Containerization | Docker |
